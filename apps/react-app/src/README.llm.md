# React App Source

Main source code for the React application built with Vite, TypeScript, and TanStack Router.

## Modules

### `apps/react-app/src/main.tsx`

Application entry point that sets up and renders the TanStack Router.

**Exports**: None (side effects only)

**Functions**:

- Creates a router instance using the generated route tree
- Renders `RouterProvider` wrapped in `StrictMode`
- Assumes `#root` element exists in HTML

**Dependencies**:

- `react` - React library
- `react-dom/client` - React DOM client API
- `@tanstack/react-router` - Router library
- `apps/react-app/src/routeTree.gen` - Auto-generated route tree (created by TanStack Router Vite plugin)

**Gotchas**:

- Uses non-null assertion (`!`) on `getElementById` - assumes root element exists
- Wraps app in `React.StrictMode` for development checks
- Route tree is auto-generated by `@tanstack/router-vite-plugin` - do not edit manually

### `apps/react-app/src/routes/__root.tsx`

Root route component that provides the layout for all routes.

**Exports**:

- `Route` - Root route configuration

**Component Features**:

- Provides navigation links (Home, About)
- Renders `<Outlet />` for child routes
- Uses inline styles for styling

**Dependencies**:

- `@tanstack/react-router` - Router library

**Gotchas**:

- Must be named `__root.tsx` for TanStack Router file-based routing
- Uses `<Outlet />` to render child routes

### `apps/react-app/src/routes/index.tsx`

Home page route that displays environment variables and demonstrates package1 utilities.

**Exports**:

- `Route` - File route configuration for `/`

**Component Features**:

- Displays app name, environment variables, and package1 utilities
- Uses inline styles for styling

**Dependencies**:

- `@tanstack/react-router` - Router library
- `package1` - Uses `add`, `capitalize`, `formatCurrency` utilities

**Environment Variables**:

- `VITE_APP_NAME` - Application name (default: 'React App')
- `VITE_API_URL` - API URL (default: 'http://localhost:3000')
- `VITE_ENV` - Environment name (default: 'development')

**Gotchas**:

- Must be named `index.tsx` to create the `/` route
- Environment variables must be prefixed with `VITE_` to be accessible in the browser

### `apps/react-app/src/routes/about.tsx`

About page route.

**Exports**:

- `Route` - File route configuration for `/about`

**Component Features**:

- Simple about page with information about the app

**Dependencies**:

- `@tanstack/react-router` - Router library

**Gotchas**:

- File name determines the route path (`about.tsx` â†’ `/about`)

### `apps/react-app/src/vite-env.d.ts`

TypeScript type definitions for Vite environment variables.

**Exports**: None (type definitions only)

**Purpose**: Provides TypeScript types for Vite-specific globals and environment variables

### `apps/react-app/src/test/setup.ts`

Test configuration - see `apps/react-app/src/test/README.llm.md` for details.

## Relationships

- `apps/react-app/src/main.tsx` creates router instance and renders `RouterProvider`
- `apps/react-app/src/routes/__root.tsx` provides the root layout and navigation
- `apps/react-app/src/routes/index.tsx` imports utilities from `package1` workspace package
- `apps/react-app/src/routes/about.tsx` provides the about page route
- All route components use Vite's `import.meta.env` for environment variables
- Route tree is auto-generated by `@tanstack/router-vite-plugin` during build/dev
- TanStack Router uses file-based routing where file names determine route paths
